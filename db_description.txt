Текстовое описание базы данных

Данная БД описывает работу автомобильного сервиса и содержит след. таблицы:
1. autos - Справочник по маркам и моделям автомобилей, индекс.
2. cities - Справочник по городам
3. client_auto - Список авто клиентов. Вынесена в отдельную таблицу на случай если клиент обслуживает несколько своих авто
4. clients - Список клиентов
5. countries - Справочник стран
6. engine_types - Справочник типов моторов, привязан триггер
7. orders - Основная таблица, перечень заказов. 
8. orders_archive - Таблица для архива заказов, наполняется процедурой
9. parts - Таблица запчастей
10. pricelist - Таблица услуг сервиса
11. staff - Список сотрудников

Триггеры и функции
1. Триггер реализует проверку ввода на предмет совмещения параметра "обьем двигателя" с типом двигателя "электрический", которые являются несовместимыми
2. Функция подсчитывает стоимость конкретонго заказа, принимая его ид в качестве параметра
3. Процедура использует транзакцию для перемещения данных в архивную таблицу, принимая в качетсве параметра дату, до которой нужно перемещать данные

Представления и запросы
1. Первое преставление дает подробную информацию о клиентах и их авто
2. Второе представление дает полную информацю о заказе, включая имя клиента, название авто и стоимось его заказа
3. Запрос выдает след выборку данных : средний возраст клиента, средний стаж сотрудников, самая популярная марка авто у клиентов и их количество
